# HyperRoot
HyperRoot is a high-performance C++ implementation designed as a modern alternative to PRoot. It provides a user-space environment that simulates a root filesystem without requiring actual superuser privileges on a Linux host.
(Note:Only supports x86_64 currently)
How do i compile hyperroot?:
1. git clone https://github.com/Ilikemycheesedrippybruhlol/HyperRoot
2. cd HyperRoot
3. cd Versions
4. g++ HyperRootv1.cpp -o hyprroot
5. Run hyperroot by ./hyprroot and ur desired command you want to run as root
